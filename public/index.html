Request Header Parser Microservice

A javascript app that reads your browser and returns a JSON object with your ip-address, language and software.

User stories:

  1. I can get the IP address, language and operating system for my browser.

Example query:

https://answart-rhp-ms.herokuapp.com/api/whoami

Output

{"ip-address":"xxx.xxx.xxx.xxx","language":"en-US","software":""Macintosh; Intel Mac OS X 10.13; rv:58.0""}
